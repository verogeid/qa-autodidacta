Semana 7: Listas y keys

Esta semana aprenderás a mostrar listas dinámicas usando el método map y la importancia de las keys.

Vamos a crear una lista de tareas.

Escribe:

import React, { useState } from 'react'

function App() {
  const [tareas, setTareas] = useState(['Aprender React', 'Practicar Hooks', 'Crear proyectos'])

  return (
    <div>
      <h1>Lista de tareas</h1>
      <ul>
        {tareas.map((tarea, index) => (
          <li key={index}>{tarea}</li>
        ))}
      </ul>
    </div>
  )
}

export default App

Explicación:

map recorre el array tareas y devuelve un elemento li por cada tarea.

La prop key ayuda a React a identificar cada elemento para optimizar actualizaciones.

Resumen:

Renderizar listas es esencial para manejar datos dinámicos.

Usar keys correctamente evita errores y mejora el rendimiento.
